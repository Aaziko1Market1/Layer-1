=== SEQUENTIAL ENRICHMENT MODE - QUICK START ===

NEW FLOW (Strict Sequential):
------------------------------
Google API → Global API → Brave Search → AI → Apollo → Hunter.io → Snov → ZeroBounce (MANDATORY)

Each step receives output from previous step!


SETUP:
------
1. Add to .env (MANDATORY):
   ZEROBOUNCE_API_KEY=your_key_here
   APOLLO_API_KEY=your_key_here
   BRAVE_SEARCH_API_KEY=your_key_here
   HUNTER_API_KEY=your_key_here
   SNOV_API_KEY=your_key_here (optional)

2. Build:
   npm run build


RUN SEQUENTIAL MODE:
--------------------
node -r dotenv/config dist/services/etl/pipeline.js \
  10 10 10 10 \
  --sequential \
  dotenv_config_path=.env

Args: extractLimit classifyLimit verifyLimit agentLimit --sequential


WHAT HAPPENS:
-------------
For each buyer:
  [1/8] Google Scraper → gets search results
  [2/8] Global API → receives Google results
  [3/8] Brave Search → receives Global results
  [4/8] AI Analysis → receives Brave results
  [5/8] Apollo → receives AI results
  [6/8] Hunter.io → receives Apollo results
  [7/8] Snov → receives Hunter results
  [8/8] ZeroBounce → verifies ALL emails (MANDATORY!)

Time: ~2 minutes per buyer
